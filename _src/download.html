Release Notes

Instructions for checking hashes and signatures is described on the Verifying Apache Software Foundation Releases page. These KEYS can be used to verify the integrity of the downloaded artifact using PGP signature (.asc).

Bleeding-Edge

You can also retrieve the source files from our Git repository and create a bleeding-edge build by typing:

git clone {{ site.data.project.source_repository_mirror }}
cd systemml
mvn clean package

Using Maven

We strongly encourage our users to configure a Maven repository mirror closer to their location.
Please read How to Use Mirrors for Repositories.

Dependencies

Add the following dependencies to your Maven pom.xml.

<dependency>
  <groupId>org.apache.systemml</groupId>
  <artifactId>systemml</artifactId>
  <version>{{ site.data.project.release_version }}</version>
</dependency>

Previous Releases

All previous releases of Apache SystemML can be obtained from archives. For Spark 1.6 support, please use Apache SystemML 0.12.
